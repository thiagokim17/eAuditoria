Esse projeto é um projeto proposto pela empresa e-Auditoria

Abrir a solution backEnd\eAuditoria\eAuditoria.sln

--BackEnd
1º Alterar a connection string no arquivo Program.cs
2º Criar no banco de dados um database com o nome 'eAuditoria'
3º executar no Package Manager Console o comando update-database
	-Para que as tabelas sejam criadas no banco de dados e consiga fazer uso da aplicação
4º Executar a aplicação e manter a mesma e execução


--Front End
5º Executar o comando 'yarn' para que as dependências sejam instaladas
6º Executar o comando 'yarn start' para inicializar a aplicação
